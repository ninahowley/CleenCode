## Test Local Edits

1. **Clone the repository**
    ```
    git clone https://github.com/ninahowley/cleencode.git
    ```
2. **Install flask**
    ```
    pip install flask
    ```
3. **Run the app!**

   Mac: ```flask --app app run```

   Windows: ```python app.py```

## Tech Stack
+ **Frontend:** Flask, HTMl, CSS, JavaScript
+ **Backend:** Python and Pandas
